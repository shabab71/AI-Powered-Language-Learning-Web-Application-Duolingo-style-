# AI-Powered-Language-Learning-Web-Application-Duolingo-style-
This project is a full-stack AI-powered language learning web application inspired by Duolingo.
The platform enables users to:
- Get Language Translation using Argos Library similar to Google Translate 
- Learn vocabulary and structured lessons
- Take interactive quizzes
- Track learning progress
- Experience real-time language translation
- Use text-to-speech (TTS) functionality
